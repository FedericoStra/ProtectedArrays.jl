## [0.1.0]

### Added

- Types `ProtectedArray`, `ProtectedMatrix`, `ProtectedVector`.
- Functions `protect`, `unprotect` to wrap and unwrap an array.

### Fixed

- Compatibility with Julia `VERSION >= v"1.8"`.
- CI.
